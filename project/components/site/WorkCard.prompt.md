**CRTR WorkCard** — portfolio tile for the showcase grid; colored field with bottom-left meta + display title, lifts with a hard shadow on hover.

```jsx
<WorkCard tone="darkGrad" meta="2025 · Brand film" title="Halftime — full reset" />
<WorkCard tone="red" meta="2025 · Identity" title="Marrow Coffee" />
```

Tones: `ink` `red` `redGrad` `darkGrad` `steel` `gold` `paper`. Use `style={{gridColumn:'span 4'}}` to size within a CSS grid.
