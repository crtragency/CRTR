**CRTR Card** — bordered paper surface that lifts with a hard offset shadow on hover; the default container for grouped content and testimonials.

```jsx
<Card><h4>Marrow Coffee</h4><p>Identity system, 2025.</p></Card>
<Card tone="ink" interactive={false} padding={32}>Dark panel</Card>
```

Tones: `paper` `ink` `red`. Set `interactive={false}` for static panels. `padding` is px.
