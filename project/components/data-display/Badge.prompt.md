**CRTR Badge** — small mono pill for status, availability and eyebrow labels.

```jsx
<Badge tone="ink" dot>Booking — May</Badge>
<Badge tone="red">New</Badge>
<Badge tone="softred">Limited</Badge>
```

Tones: `paper` `ink` `red` `softred`. `dot` adds a leading status indicator (green on ink, red elsewhere).
